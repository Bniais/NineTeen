 La frame d'animation d'affichage du combo.
 
int scoreDest
 Le score total visé
 
int combo

int flat
 Le nombre affiché de points ajoutés par le bonus flat (tend vers flatDest)
 
int flatDest
 Le nombre de points total ajoutés par le bonus flat.
 
int multi
 Le coefficient du bonus multi.
 
int sameColor
 La couleur de la ligne, vaut -1 si ligne de différentes couleurs.
